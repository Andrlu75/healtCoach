# Фаза 9: Web Console — Расширение

## Что уже есть:
- Auth (Login) + JWT store
- Dashboard (статистика клиентов)
- Clients list (таблица + фильтры + действия)
- Settings: AI, Telegram, Persona

---

## Что нужно реализовать:

### 1. ClientDetail — страница клиента
- Профиль: имя, статус, город, нормы КБЖУ (редактируемые)
- Вкладки: Питание (список приёмов пищи), Метрики (график), Чат (история)

### 2. Onboarding Editor — редактор анкеты
- Список вопросов (порядок)
- Создание/редактирование вопроса (текст, тип ответа, опции)
- Удаление

### 3. Reports — отчёты
- Список отчётов (фильтр по клиенту, типу)
- Генерация вручную
- Просмотр summary + скачивание PDF

### 4. Invites — инвайт-ссылки
- Создание инвайта
- Список существующих
- Копирование ссылки + удаление
